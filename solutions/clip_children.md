# clip_children으로 불필요한 하위 요소 잘라내기
[◀](../README.md)

패널 크기 밖에 있는 하위 요소들을 지워준다.
이게 채감은 제일 크다.

### 별로 안좋은 효과 같은데 뭐가 좋다는거?
- 알아야할 점이 생각보다 패널 밖에 렌더링 되는 안보이는 요소들이 많다.
- 왜냐하면 서로 다른 폼을 상정하고 만들었으니 크기 규격이 다른 것이다.
- 모든 폼의 크기를 동일하게 하면 채감이 잘 안될 수도 있지만 다양한 크기의 폼을 사용한다면 렉이 크게 줄어든다.
- 진짜 크기를 조금만 벗어나도 시원하게 잘라주니 패널 밖에 오브젝트 배치를 하려고 한 것이 아니라면 이건 안 쓸 이유가 없음

### 예시
```json
{
    "test_panel": {
        "type": "panel",
        "size": [320, 260],
        "clip_children": true,
        "controls": [ //이 하위 요소들은 [320, 260] 넘어가는 부분이 잘림
            ...
        ]
    }
}
```
